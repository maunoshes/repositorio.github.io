Construcción

Los pasos que seguimos para crear nuestra pagina web fueron los siguientes:

  1. Primero, instalamos Visual Studio Code y la extensión Live Server.
  2. Creamos una carpeta en nuestra computadora y la abrimos en Visual Studio Code.
  3. Dentro de la carpeta, creamos un archivo index.html con la estructura básica de una página web.
  4. Hacemos clic derecho en el archivo index.html y seleccionamos "Open with Live Server" para visualizar los cambios en tiempo real.

En esta práctica, logramos crear nuestra propia página web utilizando Visual Studio Code.
